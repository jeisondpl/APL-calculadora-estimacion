6. Tokens CSS (Variables)

Propuesta de tokens CSS para implementar el sistema de diseño como variables nativas.

:root {
  /* ── COLORES ─────────────────────────────────── */
  --color-primary-dark:    #004254;   /* Deep Teal  – Header, footer bottom */
  --color-primary:         #03657C;   /* Teal       – Nav bar, títulos, bordes */
  --color-accent:          #FBBB21;   /* Amber      – CTA, badge Profesionales */
  --color-link:            #007BFF;   /* Blue       – Links, H1 */

  --color-text:            #212529;   /* Dark Gray  – Cuerpo principal */
  --color-text-secondary:  #666666;   /* Medium Gray– Texto secundario */

  --color-text-subdued:    #868689;   /* Muted Gray – Placeholders */

  --color-surface:         #FFFFFF;   /* Fondo principal */
  --color-surface-alt:     #F8F8F8;   /* Fondo alternativo – Footer top, dropdown */

7. Reglas de Uso y Accesibilidad

ReglaDescripción
Contraste mínimoTexto claro sobre fondos oscuros: #FFFFFF sobre #004254 (ratio 10.3:1 ✓) y #FFFFFF sobre #03657C (ratio 7.8:1 ✓)
Contraste AmberTexto #004254 sobre #FBBB21: ratio 6.2:1 ✓ — usar siempre dark teal sobre amber, no blanco
Fuentes web-safe fallbackSiempre incluir Helvetica, Arial, sans-serif como fallback para NeoSansPro; Georgia, serif para NoeDisplay
Diseño flatSin sombras de caja en componentes propios de Indra. border-radius: 0 salvo botón secundario (4px) e inputs
CTA primarioUsar borde Amber grueso (4px solid #FBBB21) en acciones principales de edición
GridBootstrap cols: usar col-6 para splits 50/50 en datos de perfil
Badge activoSección activa: fondo #FBBB21, texto #03657C. Clase .active
IconografíaIconos de navegación en header: iconos SVG/font-icon blancos sobre fondo #004254
